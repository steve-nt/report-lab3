================================================================================
KEY CONCEPTS EXPLAINED IN COMMENTS:
================================================================================

FEDERATED LEARNING:
- Decentralized model training with multiple clients
- Global model distribution to local clients (regression)
- Local training on private datasets
- Federated Averaging (FedAvg) aggregation algorithm
- Sequential simulation (non-parallel local training)

PHASE 1 (PRE-TRAINING):
- Initialize models and bind datasets
- Distribute global weights to local clients
- Train each local client on their dataset
- Aggregate weights using FedAvg
- Evaluate global model on merged test data
- Track accuracy and loss across rounds

PHASE 2 (OOD DETECTION):
- Activate OOD protection mechanism
- Extract features from all neural network layers
- Project features to hyperdimensional space
- Bundle (superpose) projected features
- Compute cosine similarity between local and global models
- Filter out-of-distribution clients before aggregation

HYPERDIMENSIONAL COMPUTING:
- Use very high-dimensional vectors (e.g., 1e4 dimensions)
- Bundle operation combines information via addition
- Bind operation associates concepts via multiplication
- Projection matrices map layer outputs to hypervector space
- Cosine similarity enables robust model comparison

IMAGE PROCESSING:
- Grayscale conversion for consistency
- Contrast enhancement via weighted sum (1.5x - 0.5x)
- Sharpening kernel for edge detection
- Pixel clipping and normalization
- Batch processing with ImageDataGenerator
